wcc.footer Installation
-----------------------

To install wcc.footer using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``wcc.footer`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        wcc.footer

* Re-run buildout, e.g. with:

    $ ./bin/buildout
